**Status**: DRAFT (hazırlanıyor)

**Author**: Analysis Team

## Özet

v0.11 Lovax Language'ı kritik bir dönüm noktasına taşıyacaktır:

1. **Dosya yapısını modernize et** - Monolith `.hpp` dosyalarını modüler yapıya çevir
2. **P0 (critical) featurleri ekle** - Complex Numbers, DateTime, Regex, Better Random
3. **Ölçeklenebilirliği sağla** - Gelecek featurler kolayca eklenebilsin

Bu RFC, v0.11'deki 4 ana özellik için spesifikasyonlar içeriyor.

## PART 2: COMPLEX NUMBERS (RFC-017 Summary)

### Amaç
Lovax'a complex number support ekle: `1 + 2j`

### Spesifikasyon
```lovax
kull complex = 3 + 4j
say complex             # (3, 4)

say complex.abs()       # 5
say complex.real        # 3
say complex.imag        # 4

say complex + 1+2j      # (4, 6)
say complex * 2j        # (-8, 6)
```

### Yapılacaklar
1. **object/object.hpp**: `ComplexObject` class
2. **vm/value.hpp**: `VKind::COMPLEX` enum
3. **vm/compiler.hpp**: Arithmetic (+, -, *, /, %)
4. **modules/math_module.hpp**: cmath functions
5. **Tests**: 20+ cases

### Timeline
- **Zaman**: 10 saat
- **Tests**: 2 saat
- **Docs**: 1 saat

## PART 3: DATETIME & CALENDAR (RFC-018 Summary)

### Amaç
Lovax'a datetime support ekle

### Spesifikasyon
```lovax
use datetime

dt = datetime.now()              # Current date/time
say dt.year, dt.month, dt.day    # 2024, 7, 16
say dt.hour, dt.minute           # 15, 30

ts = 1234567890
dt2 = datetime.from_timestamp(ts)

delta = datetime.timedelta(days: 1, hours: 2)
dt3 = dt + delta  # Adding time

say datetime.strftime(dt, "%Y-%m-%d")  # "2024-07-16"
```

### Yapılacaklar
1. **object/object.hpp**: `DateTimeObject` class
2. **modules/datetime_module.hpp**: now(), from_timestamp(), timedelta(), etc.
3. **Tests**: 20+ cases

## PART 4: REGULAR EXPRESSIONS (RFC-019 Summary)

### Amaç
Lovax'a basic regex support ekle (v0.11'de ÖNEMLİ: BASAL sadece)

### Spesifikasyon
```lovax
use regex

# Match
say regex.match("hello", "h.*o")     # true
say regex.match("hello", "x.*y")     # false

# Find all
matches = regex.find_all("123-456-789", "\\d+")
# ["123", "456", "789"]

# Replace
result = regex.replace("hello world", "l+", "L")
# "heLo worLd"
```

### Yapılacaklar
1. **modules/regex_module.hpp**: match(), find_all(), replace()
2. **Basit regex engine** OR **Std::regex wrapper**?
3. **Tests**: 15+ cases

### NOT v0.11'de DİĞER
- Lookahead/lookbehind (v0.12+)
- Named groups (v0.12+)
- Flags (case-insensitive, etc. - v0.12+)

### Timeline
- **Zaman**: 10 saat (std::regex wrapper)
- **Tests**: 2 saat
- **Docs**: 1 saat
